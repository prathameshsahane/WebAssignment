Validating Web Forms


First name must be alphanumeric and contain 3-16 characters

Last name must be alphanumeric and contain 3-16 characters

Email must be valid address,eg. abc@gmail.com

Password must be alphanumeric(@,_ and - are allowed)
and between 6-20 characters

A valid phone number(11 digits)

Bio must contains only lower case letters,underscores,hyphens and be 8-50 characters
